Cookbook 2.0

About This Project

Cookbook 2.0 is a full stack web application I wrote to practice the creation of RESTful APIs and their implementation. Using a C# backend, mySQL for the database, and a react frontend, I built out the application to support creating and categorizing recipes. Long term, I am considering adding user accounts and custom recipe properties, however the next step at this time is to focus on styling. This project is an ongoing WIP, serving as a testbed of sorts where I can practice full stack development and self hosting.

Technologies used: React, C#, mySQL
