Yugioh Deck Optimizer

About This Project

A deck optimizer for the Yugioh trading card game, this is a purely JS application for now. The project allows a user to upload a deck list, define card categories, define what sort of criteria they want to find in a hand, and then test. It draws thousands of test-hands in seconds and compares them against the criteria to return a clear consistency percentage. And then the magic kicks in. The user can tell the application which cards can have their count altered and have the program check all of those variations to find the optimal changes for a specific target hand. This project was my first time using an AI-First development approach. I started it primarly to practice spec-driven development via Claude. This has resulted in a working prototype exceptionally fast, although the tradeoff at this time is some questionable code choices and a few broken areas. I'm continuing to iterate on it and as such treate this project as an active WIP.
